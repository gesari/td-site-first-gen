Thymoran Dreams

This is the landing page for the game system, which I have tentatively titled Thymoran Dreams. For newcomers, the materials are intended to be traversed in the order they are listed below, and should hopefully build on each other as they progress.

System Basics

This portion of the site reviews the basic gameplay loop for Thymoran Dreams: how to use the dice to determine the outcomes of sitautions, and the actions Players may take within the game space to influence the world around them.

Entity Basics

Building on the System Basics, this section of the site covers the actors within the game space. All PCs are entities, and their features are introduced here.

Character Creation

Once we have a basic understanding of the game space and the actors within it, we can start creating characters! This section walks players through one way the character creation process can unfold, including a fictional party and campaign premise to serve as an example.

Character Generation

A digital character creator. Currently a link to a google doc that you can use to play, but it is shared by all users currently on the site. If you'd like a copy of your own to play around with before I've implemented a personal version, let me know!

System Codex

The entire game system's reference document. Every rule should be detailed in here in some form or other.
