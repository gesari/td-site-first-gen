Start Dreaming

This landing page contains a list of primers for the Thymoran Dreams Game and accompanying Dreamscape setting: each set of links has its own column below. For newcomers, the materials are intended to be traversed in the order they are listed, and should hopefully build on each other as they progress.

Game Primer

System Basics

This portion of the site reviews the basic gameplay loop for Thymoran Dreams: how to use the dice to determine the outcomes of sitautions, and the actions Players may take within the game space to influence the world around them.

Entity Basics

Building on the System Basics, this section of the site covers the actors within the game space. All PCs are entities, and their features are introduced here.

Character Creation

Once we have a basic understanding of the game space and the actors within it, we can start creating characters! This section walks players through one way the character creation process can unfold, including a fictional party and campaign premise to serve as an example.

Setting Primer

Key Points

This section will cover ten points about the setting that will apply to any situation you find yourself in. This doesn't go deeply into the history of Thymora or the Dreamscape itself, but rather a list of things to know about how the setting itself functions. Hopefully, this should help get a feel for the Dreamscape itself before you have to worry about the specifics of any location you may find yourself in.

Fitting In

After getting an idea of how the game space itself operates, this document will help you begin to understand where your character's place is.
